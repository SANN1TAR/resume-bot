[bot.py]
Что делает: главный файл бота. Ведёт диалог с пользователем по шагам — собирает данные для резюме (ФИО, возраст, город, телефон, почта, должность, опыт, навыки, образование, о себе). После последнего шага вызывает pdf_gen.py и отправляет готовый PDF.
Связан с: pdf_gen.py (генерация PDF), .env (токен бота)
Как использовать: создай .env файл с BOT_TOKEN, запусти python bot.py
Команды:
  /start — начать заполнение резюме
  /cancel — отменить в любой момент

[pdf_gen.py]
Что делает: генерирует PDF резюме из словаря с данными пользователя. Делает красивую вёрстку: имя крупно, должность красным, контакты, разделители, секции.
Связан с: bot.py (вызывается из него), DejaVuSans.ttf (шрифт с кириллицей — если есть)
Вход: словарь с ключами name, age, city, phone, email, target_job, experience, skills, education, about
Выход: путь к временному PDF файлу в /tmp/

[requirements.txt]
Что делает: список зависимостей Python.
Как использовать: pip install -r requirements.txt

[.env.example]
Что делает: шаблон для .env файла. Скопируй в .env и вставь токен бота.

[Procfile]
Что делает: конфиг для деплоя на Railway.
